# CADU — Planos de Saúde (landing)

Clone da landing “Setor Público”, com a **mesma identidade visual e estrutura**, para adaptar o conteúdo ao contexto de **operadoras / planos de saúde**.

## Rodar local

```bash
npm install
npm run dev
```

O terminal imprime o link de rede (para abrir em outro dispositivo no mesmo Wi‑Fi).

## Export HD (para arquivo/compartilhamento)

```bash
npm run export:hd
```

Gera em `exports/hd/` (sem `node_modules`/`.next`).
